The scroller below uses CSS zoom: 1.5. Wheel deltas are integer device-pixels, but at zoom 1.5 the resulting scrollTop is in CSS pixels = deviceΔ / 1.5, which is fractional. That style is mirrored into the local DOM, so bbx-client reads the same fractional value.
